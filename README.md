# Blog

This is a Microservices practice project created using:

- [Nodejs](https://nodejs.org/) and [Express](https://expressjs.com/)
- [React](https://reactjs.org/)

It's a app where you can:

- Add posts
- Comment on posts
- Comments get moderated and rejected if contains the word 'orange'

# How to run this project locally

- Clone this repo
- Inside each folder `/client`, `/comments`, `/event-bus`, `/moderation`, `/posts` and `/query`:
  - Run `npm install`
  - Run `npm start`
- Client will run on port 3001
